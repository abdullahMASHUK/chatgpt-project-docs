# Handover Bundles

A handover bundle is a **copy/paste pack** you move into your **code repo** so Copilot can read it.

## Where to put it in code repo
Recommended:
- `code-repo/docs/_handover/`

## What Copilot should read
- `docs/_handover/_global/*`
- `docs/_handover/features/*`

## Bundles
- `bundles/HANDOVER-latest/` — current handover pack
- `bundles/HANDOVER-YYYY-MM-DD/` — optional snapshots
